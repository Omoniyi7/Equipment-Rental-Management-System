# Equipment Rental Management System

## Overview
This project is a web-based equipment rental management system designed to manage inventory and allow users to rent and return equipment efficiently.

## Features
- User authentication (Admin & User roles)
- Equipment management (CRUD operations)
- Equipment rental and return system
- Rental history tracking
- Real-time inventory updates
- Admin reporting dashboard

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
- XAMPP

## Installation
1. Copy project folder to `htdocs`
2. Start Apache & MySQL in XAMPP
3. Import `equipment_rental_db.sql` into phpMyAdmin
4. Open:
   http://localhost/equipment_rental_system/
